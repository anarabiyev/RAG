# Go

Go, also called Golang, is a statically typed, compiled programming language designed at Google by Robert Griesemer, Rob Pike, and Ken Thompson. It was publicly announced in 2009 and reached version 1.0 in 2012. Go was created to improve programming productivity at scale, with a focus on simplicity and fast compilation.

Go has built-in support for concurrency through goroutines and channels, which makes it well suited to networked services and backend systems. It includes a garbage collector, unlike Rust. Go is frequently used for cloud infrastructure, web servers, and DevOps tooling; Docker and Kubernetes are both written in Go. The language deliberately keeps its feature set small to keep code easy to read and maintain.

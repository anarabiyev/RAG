# Python

Python is a high-level, general-purpose programming language created by Guido van Rossum and first released in 1991. Its design philosophy emphasizes code readability through significant indentation. Python is dynamically typed and garbage-collected, and it supports multiple paradigms including procedural, object-oriented, and functional programming.

Python is widely used in data science, machine learning, web development, automation, and scripting. Its large standard library and ecosystem of third-party packages such as NumPy, pandas, and PyTorch make it the default choice for most machine learning and AI work. Python is an interpreted language, which makes it easy to prototype with but generally slower at raw computation than compiled languages.

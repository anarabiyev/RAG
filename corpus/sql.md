# SQL

SQL, which stands for Structured Query Language, is a domain-specific language used to manage and query data in relational databases. It was developed at IBM in the 1970s by Donald Chamberlin and Raymond Boyce, based on the relational model proposed by Edgar F. Codd. SQL became an ANSI standard in 1986.

Unlike general-purpose languages such as Python or Rust, SQL is declarative: you describe what data you want rather than how to fetch it, and the database engine decides how to execute the request. SQL is used with relational database systems like PostgreSQL, MySQL, and SQLite. Core operations include SELECT for reading data, INSERT for adding rows, UPDATE for changing them, and DELETE for removing them. Many vector databases used in RAG systems, such as pgvector, are built as extensions on top of SQL databases like PostgreSQL.
